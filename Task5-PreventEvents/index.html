Функція event.stopPropagation() дозволяє виконати подію на поточному елементі, але блокує виконання функції на всіх батьківських елементах. Для кращого розуміння, розгланемо приклад: для двох блоків, що нище, на подію onClick описані дві функції(alert-ом виводиться повідомлення). Для правого виконується функція із застосовується event.stopPropagation(), для лівого ні. Якщо клікнути по лівому блоці, то функція виконається для поточно елемента(на котрому клікнули) і ще по одному разу для кожного батьківського. У випадку з правим блоком, то функція виконається тільки один раз - для поточного елемента.

Сходження - функції preventDefaul() і stopPropagation() використовуються для блокування подій.
Відмінності - перша блокує стандартну дію браузера, друга - виконання події для батьківських елементів.
